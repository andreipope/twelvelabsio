---
title: "Update an index"
description: "Update the name of the specified index."
api: "PUT https://api.twelvelabs.io/v1/indexes/{index-id}"
---

### Path Params

<ParamField path="index-id" type="string">
  Unique identifier of the index to retrieve.
</ParamField>

### Body Params

<ParamField body="index_name" type="string">
  The name of the index.
</ParamField>

### Response Schema

The response is in JSON format.

On success, it contains the following fields:

- `message`: A human-readable string describing the result.

- `type`: Describes the type of operation (`update_index`).

On failure, it contains the following fields:

- `error_code`: The code associated with the error. See the
  [Error Codes](/reference/api-reference/error-codes) page for details.

- `message`: A human-readable string describing the error.

<ResponseExample>

```json
{
  "message": "Succesfully updated index '{index_name}'",
  "type": "update_index"
}
```

</ResponseExample>
