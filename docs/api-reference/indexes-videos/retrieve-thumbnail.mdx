---
title: "Retrieve a Thumbnail"
description: "Retrieve a thumbnail from the specified video vector"
api: "GET
  https://api.twelvelabs.io/v1/indexes/{index-id}/videos/{video_id}/thumbnail"
---

### Path Params

<ParamField path="index-id" type="string" required>
  The unique identifier of the index with which the video vector is associated.
</ParamField>

<ParamField path="video-id" type="string" required>
  The unique identifier of the video vector to retrieve.
</ParamField>

### Query Params

<ParamField path="time" required>
  Specifies the time, in seconds, at which the API service must retrieve the
  thumbnail. If you don't specify this parameter, the API service retrieves a
  thumbnail from the middle of the video.
</ParamField>

### Response Schema

The response is in JSON format.

On success, it contains the following fields:

- `data`: An object that contains the thumbnail.

  - `thumbnail`: A string representing the URL of the thumbnail. Note that the
    URL expires in one hour.

- `type`: Describes the type of operation (`get_video_thumbnail`).
