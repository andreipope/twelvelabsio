---
title: "List Search Engines"
description: "List the search engines that are available in your account."
api: "GET https://api.twelvelabs.io/v1/engines"
---

### Response Schema

The response is in JSON format. It contains the following fields:

- `data`: An array that contains the engines.

- `type`: Describes the type of operation (`engine_list`).
