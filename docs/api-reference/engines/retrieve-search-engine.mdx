---
title: "Retrieve a Search Engine"
api: "GET https://api.twelvelabs.io/v1/engines/:engine-id"
---

### Path Params

<ParamField path="engine_id" type="string">
  The identifier of the engine to retrieve. The following search engines are
  available: - `marengo2` - `marengo` For details about the features of each
  search engine, see the [Search Engines](/concepts/search-engines) page.
</ParamField>

### Response Schema

The response is in JSON format.

On success, it contains an `engine` object.

On failure, it contains the following fields:

- `error_code`: Represents the code associated with the error. See the
  [Error Codes](/reference/api-reference/error-codes) page for details.

- `message`: A human-readable string describing the error.
