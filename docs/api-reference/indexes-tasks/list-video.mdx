---
title: "List video indexing tasks"
description:
  "Returns a list of the video indexing tasks in your account. The API returns
  your video indexing tasks sorted by creation date, with the newest at the top
  of the list."
api: "GET https://api.twelvelabs.io/v1/indexes/tasks"
---

### Query Params

<ParamField query="page" required>
  A number that identifies the page to retrieve. **Default**: `1`.
</ParamField>

<ParamField query="page_limit" required>
  The number of items to return on each page. **Default**: `10`. **Max**: `100`.
</ParamField>

<ParamField query="sort_by" type="string" required>

The field to sort on. The following options are available:

- `updated_at`: Sorts by the date and time, in the
  [RFC 3339](https://datatracker.ietf.org/doc/html/rfc3339) format, that the
  video indexing task was updated.
- `created_at`: Sorts by the date and time, in the
  [RFC 3339](https://datatracker.ietf.org/doc/html/rfc3339) format, that the
  video indexing was created. **Default**: `created_at`.

</ParamField>

<ParamField query="sort_option" type="string" required>
  The sorting direction. The following options are available: asc and desc.
  **Default**: `desc`.
</ParamField>

<ParamField query="_id" type="string" required>
  Filter by the unique identifier of a video indexing task.
</ParamField>

<ParamField query="index_id" type="string" required>
  Filter by the unique identifier of an index.
</ParamField>

<ParamField query="filename" type="string" required>
  Filter by filename.
</ParamField>

<ParamField query="duration" type="float" required>
  Filter by duration. Expressed in seconds.
</ParamField>

<ParamField query="width" type="integer" required>
  Filter by width.
</ParamField>

<ParamField query="height" type="integer" required>
  Filter by height.
</ParamField>

<ParamField query="created_at" type="string" required>
  Filter by the date and time, in the [RFC
  3339](https://datatracker.ietf.org/doc/html/rfc3339) format, that the video
  indexing task was created.
</ParamField>

<ParamField query="updated_at" type="string" required>
  Filter by the date and time, in the [RFC
  3339](https://datatracker.ietf.org/doc/html/rfc3339) format, that the video
  indexing task was created.
</ParamField>

<ParamField query="estimated_time" type="string" required>
  Filter by the date and time, in the [RFC
  3339](https://datatracker.ietf.org/doc/html/rfc3339) format, that the video
  indexing task was created.
</ParamField>

### Response Schema

The response is in JSON format.

On success, it contains the following fields:

- `data`: An array that contains up to `page_limit` tasks.

- `page_info`: An object that provides information about pagination. It has the
  following fields:

  - `limit_per_page`: The maximum number of items on each page.
  - `page`: The page you retrieved.
  - `total_page`: The total number of pages.
  - `total_result`: The total number of results.

- `type`: Describes the type of operation (`index_task_list`).

On failure, it contains the following fields:

- `message`: A human-readable string describing the error.

- `error_code`: Represents the code associated with the error. See the
  [Error Codes](/reference/api-reference/error-codes) page for details.

<Note>
  When using this endpoint, you can set up webhooks to receive notifications.
  For details, see the [Webhooks](/guides/webhooks) section.
</Note>
